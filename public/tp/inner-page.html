Aplicaciones

Estas son las aplicaciones que se le pueden hacer a las cajas de carton.

Cajas de cartón corrugado compuesta por 4 caras y aletas inferiores y superiores, sirven para contener, proteger, almacenar y transportar productos. El cierre se hace con cinta adhesiva, pegante o grapa.

  • Para llevar bebidas.
  • Para realizar trasteos.
  • Para llevar alimentos.
  • Para cajas de pizza.
